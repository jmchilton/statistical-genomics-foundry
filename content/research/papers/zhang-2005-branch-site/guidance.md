> Attention-directing, not conclusion-directing. Answer each from the source; if the source doesn't

## Must capture

- The **null distribution of the branch-site LRT**: does the paper state it is a **50:50 mixture**
  of a point mass at 0 and χ²₁ ? What critical value does it recommend at α=0.05 — does it state
  **2.71**? Does it ALSO recommend using χ²₁ (critical value 3.84) as a **conservative** guard
  against model-assumption violations? Capture BOTH recommendations precisely. (A skill flattens
  this to "2.71 NOT 3.84"; report the paper's actual, possibly two-sided, advice.)
- Type-I error / power under **post-hoc / a-posteriori foreground specification** vs pre-specified
  foreground: does the paper quantify inflation? Capture any exact number or state none is given.
- **BEB** (Bayes Empirical Bayes) for identifying positively selected sites: what posterior-
  probability values does the paper use/recommend? Quote exact numbers if stated.
