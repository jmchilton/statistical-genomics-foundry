# Guidance — smith-2015-absrel (paper)

> Attention-directing. Answer from the source; if unaddressed, say so. Backs aBSREL — the HyPhy
> branch test for episodic selection with **built-in multiple-testing correction** across branches;
> the recommended remedy when foreground branches are NOT pre-specified (avoids the post-hoc
> foreground inflation the branch-site test suffers).

## Must capture
- The **exact question** aBSREL answers (which branches show episodic selection) and how it fits an
  **adaptive** number of ω rate classes per branch (vs a fixed 2- or 3-class model). State precisely.
- The **multiple-testing correction** it applies across branches (e.g. Holm–Bonferroni) — capture
  the method name and any threshold. This is the key claim: it corrects for testing many branches.
- Test statistic + significance threshold recommended; whether testing ALL branches vs a
  pre-specified set changes the correction.
- The paper's stated advantage over the codeml branch-site test for exploratory (no a-priori
  foreground) analyses.
