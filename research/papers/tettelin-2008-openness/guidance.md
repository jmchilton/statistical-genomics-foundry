# Guidance — Tettelin 2008 (bacterial pan-genome openness)

Citation: Tettelin H, Riley D, Cattuto C, Medini D. 2008. "Comparative genomics: the bacterial
pan-genome." *Curr Opin Microbiol* 11(5):472–477. DOI 10.1016/j.mib.2008.09.006.

License posture: Elsevier, all-rights-reserved → **own-words; verbatim only for functional
strings** (equation forms, exponent symbols, numeric thresholds).

Questions (direct attention, do not confirm conclusions):

1. Does this paper introduce a power-law / Heaps'-law fit to pan-genome growth? Capture the exact
   functional form (e.g. n = k·N^γ or new-genes ∝ N^(−α)) and the EXACT symbol used for the
   exponent verbatim.
2. What is the decision rule for OPEN vs CLOSED pan-genome in terms of that exponent? Quote the
   inequality verbatim (e.g. "open if α ≤ 1"). We must know whether the rule is stated on α or on
   (1−α), and the exact boundary value.
3. Does the paper state any minimum number of genomes required to estimate the exponent reliably?
   Quote if present; note if absent.
4. Does it name specific species as open (e.g. *E. coli*, *S. pneumoniae*) vs closed (e.g.
   *B. anthracis*)? Capture the examples verbatim.
5. Does it distinguish "core," "dispensable," and any finer partition classes? Record the exact
   class vocabulary used.

Must-extract functional strings: the growth-law functional form + exponent symbol; the open/closed
inequality + boundary value; any stated minimum genome count; the named open/closed species.
